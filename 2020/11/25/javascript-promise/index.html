Promise.prototype.catch

  1. 可以捕获 Promise 参数函数和 then 内的错误

  2. 不能捕获 Promise 参数函数和 then 内的任何异步函数的错误

  3. 如果 then 没有 onReject 函数，则会触发catch。参数为reject传入的值或 throw 的值

  4. resolve 后的 throw Error 会被忽略
